# GridGuessr — AI Anime GIF Prompts

Two GIFs per moment:
- **Action GIF** (`gif`) — shown on the puzzle card. The actual moment in motion.
- **Winner GIF** (`winnerGif`) — shown on the round-result reveal. The driver celebrating.

## Style Guide (apply to every prompt)

```
Anime cel-shaded illustration, 90s sports anime aesthetic (think Capeta / Initial D
for cars, Slam Dunk / Hajime no Ippo for celebration scenes), bold ink lines,
saturated colors, dramatic motion lines / speed lines, dynamic camera angle,
seamless loop, 16:9, 3-4 second cycle.
```

Append the style guide string to every per-moment prompt below. Generate with
Sora / Runway Gen-3 / Pika / Kling, or generate the still in Midjourney + animate
in Runway. Drop the resulting URL into `MEDIA[id].gif` or `MEDIA[id].winnerGif`
inside `grid_guessr.html`.

## How to wire generated GIFs into the game

In `grid_guessr.html`, find the `MEDIA` object and replace the `null`s:

```js
const MEDIA = {
  'abu-dhabi-2021': {
    gif: 'https://your-cdn.com/abu-dhabi-2021-anime.gif',
    winnerGif: 'https://your-cdn.com/abu-dhabi-2021-winner.gif',
    youtube: 'https://www.youtube.com/watch?v=ACTUAL_VIDEO_ID'  // optional override
  },
  // ...
};
```

The game will pick up the new URLs on next page load. No JS changes needed.
